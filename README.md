# Protein Antidote GAN (TA-GAN)

Этот проект представляет собой генеративно-состязательную сеть (GAN) для создания аминокислотных последовательностей антитоксинов, обусловленных эмбеддингами токсинов. Система использует предобученную модель **ESM-2** для извлечения признаков токсинов и **Transformer-архитектуру** для генерации и дискриминации последовательностей.

## Основные особенности
- **Conditional GAN:** Генерация антидота напрямую зависит от структуры/эмбеддинга входного токсина.
- **ESM-2 Embeddings:** Использование SOTA-модели для представления белков.
- **WGAN-GP:** Обучение с использованием Wasserstein Loss и Gradient Penalty для стабильности.
- **Gumbel-Softmax:** Позволяет проводить дифференцируемое обучение на дискретных токенах (аминокислотах).
- **EMA (Exponential Moving Average):** Сглаживание весов генератора для повышения качества инференса.

## Структура проекта

### Подготовка данных
- `prepare_pairs.py`: Синхронизация и фильтрация пар токсин-антитоксин из FASTA-файлов.
- `precompute_toxin_embeddings.py`: Предварительное вычисление эмбеддингов токсинов через ESM для ускорения обучения.
- `dataset.py`: PyTorch Dataset для загрузки пар данных.

### Архитектура моделей
- `generator.py`: Transformer-декодер с предиктором длины.
- `discriminator.py`: Transformer-энкодер со спектральной нормализацией.
- `esm_utils.py`: Интерфейс для взаимодействия с библиотекой `fair-esm`.

### Обучение и утилиты
- `train.py`: Основной цикл обучения, логирование метрик и сохранение чекпоинтов.
- `losses.py`: Реализация WGAN-GP и Cross-Entropy потерь.
- `metrics.py`: Оценка качества (KL-дивергенция, разнообразие n-грамм, точность длины).
- `ema.py`: Реализация механизма экспоненциального скользящего среднего.
- `config.py`: Все гиперпараметры, пути к файлам и настройки словаря.
